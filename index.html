AutoReason

A rule-based autonomous driving simulation designed for learning, transparency, and demonstration.

AutoReason uses simple if–then logic to control a virtual car. Instead of black-box AI models, every decision is predictable, explainable, and easy to understand. This project demonstrates how autonomous behavior can be built using basic reasoning rules.

How to Use

Path Creator Mode

  • Move the camera by dragging the mouse while pressing middle mouse button.
  • Use the scrollwheel to zoom in and out.
  • Right click to change the block rotation.
  • Click green button to place the starting block.
  • Construct the path using straight and curve blocks given.
  • After the path is finished click the red button and place the end block.
  • If you are satisfied with the path click the car button.

Simulation Mode

  • Move the mouse while left clicking to move the camera.
  • Right click to toggle guide lines which shows what AI sees.
  • Press R at any time to create a new simulation.
  • You can observe the AI ouputs from the control layout on top right of the corner.
